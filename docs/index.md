# degirum-face Documentation

High-performance face recognition and tracking library for images and video. Built for production deployments with support for CPU and edge AI accelerators.

## Getting Started

**New to degirum-face?**

1. **[Install the package](getting-started/installation.md)** - Get up and running
2. **[Learn basic concepts](getting-started/basic-concepts.md)** - Understand core terminology
3. **Choose your path:**
   - [Face Recognizer](guides/face-recognizer/overview.md) - Process images and batches
   - [Face Tracker](guides/face-tracker/overview.md) - Monitor video streams in real-time
